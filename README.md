ExpenseAI - Smart Expense Tracker
An AI-powered expense tracking application with intelligent insights, budget management, and financial reports.

Features
User Authentication - Secure sign up/sign in with email
Transaction Management - Track income and expenses with categories
Category Management - Customizable categories with icons and colors
Budget Tracking - Set budgets per category or overall
Interactive Charts - Visual breakdowns of spending patterns
AI Insights - Smart financial health score and recommendations
PDF Reports - Export monthly financial reports
Dark Mode - Toggle between light and dark themes
Mobile Responsive - Works on all screen sizes
Web App
The app is already built as a web application. To run it locally:

npm install
npm run dev
